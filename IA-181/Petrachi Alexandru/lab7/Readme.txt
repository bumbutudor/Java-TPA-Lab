In lucrarea data am realizat un program care foloseste colectiile java. In cazul dat am folosit colectia ArrayList, care este o lista a carei dimensiuni
o putem defini. Am scris clasa si constructorul care creaza un obiect, care este un ArrayList. Pentru a atribui valori la lista noasta am facut doua
functii, una atribuie valori random(biblioteca random din java) si una care preia valori manual(de la utilizatori). Lucrul cu colectiile 
este mai usor, deoarece acestea au functii implicite, cum ar fi get, add, sort(nu toate) etc. Respectiv face lucrul cu colectiile mai usor.
