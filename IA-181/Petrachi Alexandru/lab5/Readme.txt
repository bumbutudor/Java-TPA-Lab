In laboratorul dat am realizat un program cu o clasa care creaza ca obiect un stack cu dimensiunea fixa, care o dam noi. Respectiv
avem functie de introducere - 'push' si de scoatere - 'pop'. Ca sarcina am avut de creat si exceptii pentru aceste actiuni. Cum ar fi
exceptia care ne anunta ca stack-ul este deja plin si nu mai putem introduce in el nimic. Sau cand dorim sa scoatem un numar si acesta lipseste,
 programul ne anunta ca stack-ul este gol. Usurinta de a lucra cu exceptiile consta in faptul ca programul nu se termina, ca in cazul erorilor, 
ci isi continua executia.
